# 덕아웃 노트

모바일 우선 KBO 마이팀 다이어리 웹앱입니다.

## 포함 기능
- 마이팀 설정
- 화이트 / 나이트 모드
- 팀별 키 컬러 적용
- 오늘의 마이팀 경기
- 전체 순위 접이식 보기
- 최근 5경기 / 10경기 달력형 보기
- 날짜 클릭 시 승리투수 / 패전투수 / 세이브 / 홀드 / 결승타 정보 표시
- 템플릿 / 자유 기록
- GPT 욕 순화 기능
- 기록 미리보기

## 데이터 소스
- KBO 공식 영문 스코어보드 / 팀 순위
- 네이버 검색 기반 상세 정보 fallback

## Vercel 환경변수
- `OPENAI_API_KEY`

## 실행
정적 프론트 + Vercel Serverless Functions 구조입니다.
